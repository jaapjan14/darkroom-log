# Changelog

## 1.0.0 — 2026-04-12

Initial release.

### Features
- Immich integration for negative search and metadata pull
- Per-image print session logging
- Single grade and split grade support with editable grade labels
- Mobile-first UI with desktop two-column layout
- Password authentication
- Edit title and edit session support
- Paper dropdown with custom entry
- Dodge/burn notes
- Delete print and delete session
- Safelight-inspired dark UI
